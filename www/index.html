보드게임 박스 상자 탑 쌓기

보드게임을 플레이한 후 [새 게임 기록하기] 버튼을 통해 기록을 남기세요. 게임을 검색하면 커버 이미지와 난이도가 자동으로 연동됩니다.

기록이 완료되면 해당 보드게임 상자의 실제 부피와 난이도(두께)에 맞춰 왼쪽 영역에 상자가 차곡차곡 위로 쌓여 탑이 올라갑니다! 누적 높이를 통해 내 성장을 확인하세요.

🧸

나만의 미플 프렌즈로 탑 꾸미기

상자 탑의 높이(cm)가 높아질 때마다 개성 가득한 30가지의 **보드게임 프렌즈 캐릭터**가 순차적으로 해금됩니다.

해금된 캐릭터를 더블클릭하거나 상세 스토리 도감에서 소환 버튼을 누르면 왼쪽 상자 탑 근처에 캐릭터 미플이 소환됩니다! 미플을 마우스나 터치로 드래그하여 나만의 탑을 예쁘게 디자인해 보세요.

🎯

월간 빙고 미션 달성

매월 새롭게 갱신되는 4x4 빙고판이 열립니다. 빙고판에 적힌 게임을 플레이하여 기록하면 해당 미션 칸이 완료 도장으로 채워집니다.

빙고 미션을 달성하고 하단의 [보상 받기] 버튼을 누르면 전설적인 '골든 메카 미플 🤖' 캐릭터가 스페셜 보상으로 해금됩니다!

🎲

실전용 보드게임 플레이 도구

보드게임을 시작하거나 플레이할 때 하단의 [도구] 탭을 활용해 보세요.

  • 선 뽑기: 이름을 적어 돌리면 선 플레이어를 골라줍니다.
  • 주사위 굴리기: 4면체~20면체 주사위를 무작위로 굴립니다.
  • 사다리 타기: 간편하게 사다리를 만들어 술래/벌칙을 지목합니다.
